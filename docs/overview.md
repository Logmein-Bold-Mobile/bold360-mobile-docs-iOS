## About
**Bold mobile SDK for iOS** is a native iOS SDK, intended to be used by Apps developed for iOS devices usage.   
Native SDK ensures usability by native applications and benefits from the operating system software features. 

## Features
Create and integrate AI and live chats to run under your App.
Customize your chat look to meet your need.

## Intended Audience
This guide assumes that you have the necessary iOS development skills to build and compile advanced applications using Xcode.
